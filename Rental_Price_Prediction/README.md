# Rental Price AirBnb Prediction

This project aims to predict the rental price of AirBnb listings based on various features such as location, room type etc. A Linear Regression model was used for prediction. The results that the model was able to achieve are R^2 = 0.1438. Such a low result is due to the fact that there are a lot of outliers in the dataset, and the linear model cannot cope with nonlinear dependencies.

| Metrics | Value |
| --- | --- |
| **MAE** | 59.185593 |
| **MSE** | 27014.250013 |
| **RMSE** | 164.360123 |
